## Props를 사용하는 방법

### Props와 state
- Props와 state는 리액트에서 데이터를 다룰 때 사용하는 개념이다.
- Props는 부모 컴포넌트가 자식 컴포넌트에게 데이터를 넘겨줄 때 사용한다.<br/>
