# 트리 알고리즘
## Balanced Tree
### 이진트리검색의 문제점
- 트리의 좌우 균형이 맞지 않으면 비효율적이다
### Balncde Tree
- 삽입과 삭제시 필요하면 스스로 균형을 유지한다
- AVL Tree, 2-3(-4) Tree, Red-Black Tree, B-tree 등등을 이용해서 균형을 유지함
- 항상 O(logN)의 검색성능
